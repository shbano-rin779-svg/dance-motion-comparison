# dance-motion-comparison
Dance Motion Comparison

概要

Dance Motion Comparison は、ダンス動画から骨格情報を抽出し、目標とするダンサーと学習者の動きの違いを可視化するダンス学習支援システムです。

ダンスの上手さは個性や表現力を含むため、単純に数値化することは困難です。一方で、多くのダンサーは「上手い人の動きを観察し、それに近づこうとする」ことで成長しています。

本システムは、コンピュータビジョン技術を用いて両者の動きを比較し、「どこがどのように違うのか」を分かりやすく提示することで、効率的な練習を支援することを目的としています。

⸻

背景

ダンス練習では、動画を撮影して振り返ることが一般的ですが、

* 自分の動きがどこで異なっているのか分からない
* 動きが小さいのか、緩急が足りないのか判断できない
* 上級者との違いを客観的に把握しにくい

といった課題があります。

そこで、映像解析によってダンサー同士の動きを比較し、改善ポイントを提示するシステムを開発しました。

⸻

主な機能

1. 人物検出・骨格推定

* YOLOによる人物検出
* MediaPipe Pose Landmarkerによる骨格推定
* 骨格座標の時系列データ化

2. 動作特徴量の抽出

各部位について以下の特徴量を計算します。

* 軌道の大きさ
* 移動距離
* 速度
* 速度のばらつき（緩急）
* ジャーク（滑らかさ）
* 停止時間割合

3. ダンサー比較

目標ダンサーと学習者について、

* 動きの大きさ
* 緩急
* 軌道
* 関節角度

を比較し、差分を可視化します。

4. Difference Timeline

動画全体の中から、

* 大きな角度差
* 軌道の違い
* 緩急の違い

が発生している時間帯を自動で抽出します。

5. Pose Snapshot

任意の時刻で動画を停止し、

* 目標ダンサー
* 学習者

の姿勢を比較します。

角度差が大きい関節はハイライト表示され、改善すべき部位を直感的に確認できます。

6. Webアプリ

Streamlitを用いてWebアプリ化しています。

ユーザーは動画をアップロードするだけで、

* 骨格推定
* 動作解析
* 差分比較
* フィードバック生成

を実行できます。
